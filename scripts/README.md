## Scripts Overview

### R Scripts
- `did_cet1_analysis.R`  
  Difference-in-differences analysis of CET1 ratios around EU climate policy announcements.

- `robustness_green_deal_2019.R`  
  Robustness check using the 2019 European Green Deal as the policy event.

### Python Scripts
- `balance_sheet_dynamics.py`  
  Constructs bank-level panel data and analyzes asset growth and leverage dynamics.

- `asset_growth_leverage.py`  
  Regression analysis linking balance-sheet adjustments to capital ratios.
